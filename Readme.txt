В этом репозитории стоит хранить данные, которые в репозиторий KEDR нет смысла класть (и делать общедоступными), но которые при этом могут быть полезными при разработке KEDR. 

Это могут быть скрипты и пр. для ночных запусков, задачи для тренировки новых участников проекта, вспомогательная документация и пр.

Данный репозиторий не предполагается делать общедоступным, т.е. выкладывать в проект KEDR на BerliOS Developer и т.д. Все общедоступные вещи стоит хранить в репозитоии KEDR, а не здесь.
